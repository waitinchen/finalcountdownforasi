# 📝 更新開發者指南內容

## 📍 位置

開發者指南組件位於首頁右下角，是一個可展開/收起的浮動面板。

## 🔧 如何更新內容

### 編輯組件文件

編輯 `app/components/DeveloperGuide.tsx` 文件來更新內容。

### 當前結構

組件包含以下部分：

1. **關於本專案** - 專案介紹
2. **技術架構** - 使用的技術棧
3. **專案結構** - 目錄結構說明
4. **快速開始** - 安裝和運行指令
5. **API 端點** - API 文檔
6. **數據結構** - 數據類型定義
7. **部署** - 部署說明
8. **聯繫信息** - 底部聲明

### 修改示例

```tsx
{/* 介紹 */}
<section>
  <h4 className="text-neon-blue text-sm font-medium mb-2 tracking-wide uppercase">
    關於本專案
  </h4>
  <p className="text-text-primary text-sm leading-relaxed">
    {/* 在這裡修改內容 */}
    你的新內容...
  </p>
</section>
```

### 添加新章節

```tsx
{/* 新章節 */}
<section>
  <h4 className="text-neon-blue text-sm font-medium mb-2 tracking-wide uppercase">
    章節標題
  </h4>
  <p className="text-text-primary text-sm leading-relaxed">
    章節內容...
  </p>
</section>
```

## 🎨 樣式類別

組件使用與主視覺相同的深空藍科技風格：

- `text-neon-blue` - 霓光藍色文字
- `bg-black/40` - 半透明黑色背景
- `border-neon-blue/20` - 霓光藍邊框
- `backdrop-blur-lg` - 毛玻璃效果
- `text-shadow-glow` - 發光文字效果

## 📱 響應式設計

組件已優化，在移動設備上會自動調整寬度：
- 桌面：固定寬度 384px (w-96)
- 移動：最大寬度為視窗寬度減去邊距

## 💡 提示

1. **保持簡潔**：內容應該簡潔明了，易於閱讀
2. **使用代碼塊**：技術內容使用 `font-mono` 類別
3. **分組內容**：使用 `<section>` 標籤分組相關內容
4. **保持風格一致**：使用相同的顏色和字體大小

## 🔄 更新流程

1. 編輯 `app/components/DeveloperGuide.tsx`
2. 保存文件
3. 開發服務器會自動重新載入
4. 在瀏覽器中查看效果

---

**注意**：由於 `.docx` 文件是二進制格式，無法直接讀取。請將文檔內容手動複製到組件中，或使用文檔轉換工具先轉換為文本格式。
